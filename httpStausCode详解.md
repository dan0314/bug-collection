# Http Status Code 网页服务器HTTP响应状态的3位数字代码

<table>
    <tr>
        <th>1xx（临时响应）</th>
        <th colspan="2">表示临时响应并需要请求者继续执行操作的状态代码。</th>
    </tr>
    <tr>
        <td>100</td>
        <td>Continue（继续）</td>
        <td>请求者应当继续提出请求。 服务器返回此代码表示已收到请求的第一部分，正在等待其余部分。</td>
    </tr>
    <tr>
        <td>101</td>
        <td>Switching Protocols（切换协议）</td>
        <td>请求者已要求服务器切换协议，服务器已确认并准备切换。</td>
    </tr>
</table>
